# 地獄の門で花と遊ぶ（MVP02）

花を置く。  
時を巡らせる。  
少しずつ、心をほどいていく。

---

## 概要
ロダンの「地獄の門」をモチーフに、  
花を配置し、時間の変化とともに作品を作るアプリのMVPです。

---

## 機能（MVP02）

- 花の配置
- 移動
- 回転
- サイズ変更
- 前面・背面の調整
- 時の巡り（状態変化）
- 作品の保存

---

## 技術構成

- Next.js
- TypeScript
- Tailwind CSS
- react-konva

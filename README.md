# 地獄の門で花と遊ぶ（MVP02）

> 花で、心をほどくアプリ。

花を置く。  
時を巡らせる。  
地獄の門に、やさしさを重ねていく。
---

## 概要
ロダンの「地獄の門」をモチーフに、  
花を配置し、時間の変化とともに作品を作るアプリのMVPです。

---

## 機能（MVP02）

- 花の配置
- 移動
- 回転
- サイズ変更
- 前面・背面の調整
- 時の巡り（状態変化）
- 作品の保存

---

## 技術構成

- Next.js
- TypeScript
- Tailwind CSS
- react-konva

---

## 次のステップ（MVP03）

- 花材選択機能
- 揺らぎ演出（時間の表現）
